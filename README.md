## Image Optimization Tips

1. **Format**: Use JPEG for photos, PNG for logos/icons, SVG for scalable icons
2. **Size**: Optimize images for web (use tools like TinyPNG or ImageOptim)
3. **Dimensions**: Follow the recommended sizes mentioned in the main README
4. **Alt Text**: All images in the HTML already have descriptive alt text
